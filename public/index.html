Wealth Tracker API

An overview of available endpoints

GET /users

Retrieve all users.

Sample Response
[{ "id": "abc123", "name": "Femi Ola", "username": "Femi", "email": "olumofemi@soezontech.com", "address": { "street": "Panatella Drive", "suite": "Apt. 556", "city": "Calgary", "zipcode": "T3J 3W2" } }]

GET /users/{id}

Retrieve a specific user by ID.

Sample Response
{ "id": "abc123", "name": "Femi Ola", "username": "Femi", "email": "olumofemi@soezontech.com", "address": { "street": "Panatella Drive", "suite": "Apt. 556", "city": "Calgary", "zipcode": "T3J 3W2" } }

POST /users

Create a new user.

Sample Payload
{ "name": "Femi Ola", "username": "Femi", "email": "olumofemi@soezontech.com", "address": { "street": "Panatella Drive", "suite": "Apt. 556", "city": "Calgary", "zipcode": "T3J 3W2" } }

PUT /users/{id}

Update an existing user.

Sample Payload
{ "name": "Updated Name", "username": "UpdatedUsername", "email": "updatedemail@example.com", "address": { "street": "New Street", "suite": "New Suite", "city": "New City", "zipcode": "12345" } }

DELETE /users/{id}

Delete a user.

GET /income

Retrieve all income records.

Sample Response
[{ "id": "inc-001", "wages": 1400, "secondaryIncome": 700, "interest": 120, "supportPayment": 40, "others": 100 }]

GET /income/{id}

Retrieve a specific income record by ID.

Sample Response
{ "id": "inc-001", "wages": 1400, "secondaryIncome": 700, "interest": 120, "supportPayment": 40, "others": 100 }

POST /income

Create a new income record.

Sample Payload
{ "wages": 1400, "secondaryIncome": 700, "interest": 120, "supportPayment": 40, "others": 100 }

PUT /income/{id}

Update an existing income record.

Sample Payload
{ "wages": 1500, "secondaryIncome": 750, "interest": 130, "supportPayment": 45, "others": 110 }

DELETE /income/{id}

Delete an income record.

GET /expenses

Retrieve all expense records.

Sample Response
[{ "id": "exp-001", "name": "Monthly Expenses", "items": { "HouseRent": 700, "LightBill": 400, "GasBill": 300, "Groceries": 200, "Others": 100 }, "total": 1700 }]

GET /expenses/{id}

Retrieve a specific expense record by ID.

Sample Response
{ "id": "exp-001", "name": "Monthly Expenses", "items": { "HouseRent": 700, "LightBill": 400, "GasBill": 300, "Groceries": 200, "Others": 100 }, "total": 1700 }

POST /expenses

Create a new expense record.

Sample Payload
{ "name": "Monthly Expenses", "items": { "HouseRent": 700, "LightBill": 400, "GasBill": 300, "Groceries": 200, "Others": 100 }, "total": 1700 }

PUT /expenses/{id}

Update an existing expense record.

Sample Payload
{ "name": "Updated Expenses", "items": { "HouseRent": 750, "LightBill": 420, "GasBill": 320, "Groceries": 220, "Others": 110 }, "total": 1820 }

DELETE /expenses/{id}

Delete an expense record.
